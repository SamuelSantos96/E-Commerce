Install Node.js and Visual Studio Code

Visual Studio Code -> Open Folder -> Project's Folder -> Terminal:

    npm install express-generator -g

    express E-Commerce --hbs

    cd E-Commerce

    npm install

    npm start   //Starts Server

    Ctrl + C    //Stops Server
